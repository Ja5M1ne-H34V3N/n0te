### 一、 体系结构与基础 (Architecture & Basics)

|缩写|英文全称|中文全称|核心用途与含义|
|:--|:--|:--|:--|
|**OSI**|Open Systems Interconnection|开放式系统互联参考模型|标准化的网络7层架构（物数网传会表应），理论基础。|
|**TCP/IP**|Transmission Control Protocol/Internet Protocol|传输控制协议/网际协议|实际网络通用的协议簇，支撑互联网运行的核心规则。|
|**LAN**|Local Area Network|局域网|覆盖单一建筑或园区（如校园、企业），高速共享。|
|**WAN**|Wide Area Network|广域网|跨城市、国家甚至全球的网络，互联网是最大的WAN。|
|**MAN**|Metropolitan Area Network|城域网|覆盖一座城市，整合多个局域网。|
|**PAN**|Personal Area Network|个人局域网|覆盖10米左右，连接个人设备（如蓝牙）。|

### 二、 物理层 (Physical Layer)

|缩写|英文全称|中文全称|核心用途与含义|
|:--|:--|:--|:--|
|**FDM**|Frequency Division Multiplexing|频分复用|划分不同频率子信道并行传输（如广播电视）。|
|**TDM**|Time Division Multiplexing|时分复用|划分不同时间片轮流传输（如数字电话）。|
|**CDM**|Code Division Multiplexing|码分复用|使用不同编码序列并行传输（3G/4G核心技术）。|
|**DWDM**|Dense Wavelength Division Multiplexing|密集波分复用|光纤通信技术，一根光纤传多波长信号，大幅扩容。|
|**SNR**|Signal-to-Noise Ratio|信噪比|信号功率与噪声功率之比，越高误码率越低。|

### 三、 数据链路层 (Data Link Layer)

| 缩写          | 英文全称                                                   | 中文全称          | 核心用途与含义                    |
| :---------- | :----------------------------------------------------- | :------------ | :------------------------- |
| **CRC**     | Cyclic Redundancy Check                                | 循环冗余校验        | 差错检测技术，判断数据传输是否出错。         |
| **GBN**     | Go-Back-N                                              | 后退N帧协议        | 出错后重传该帧及后续所有帧（累积确认）。       |
| **SR**      | Selective Repeat                                       | 选择重传协议        | 出错后仅重传出错的那一帧（效率更高）。        |
| **CSMA/CD** | Carrier Sense Multiple Access with Collision Detection | 载波监听多路访问/冲突检测 | **有线**以太网防冲突机制（边发边听，冲突停发）。 |
| **CSMA/CA** | Carrier Sense Multiple Access with Collision Avoidance | 载波监听多路访问/冲突避免 | **无线**局域网防冲突机制（预约信道，避免冲突）。 |
| **MAC**     | Media Access Control                                   | 介质访问控制        | 硬件地址，设备的唯一物理标识。            |
| **VLAN**    | Virtual LAN                                            | 虚拟局域网         | 逻辑划分广播域，隔离风暴，提升安全。         |
| **PPP**     | Point-to-Point Protocol                                | 点对点协议         | 拨号上网常用，支持身份验证的点对点链路。       |

### 四、 网络层 (Network Layer)

| 缩写          | 英文全称                               | 中文全称      | 核心用途与含义                   |
| :---------- | :--------------------------------- | :-------- | :------------------------ |
| **IP**      | Internet Protocol                  | 网际协议      | 核心协议，负责寻址和路由。             |
| **IPv4/v6** | Internet Protocol version 4/6      | IP协议版本4/6 | v4地址枯竭（32位），v6海量地址（128位）。 |
| **ARP**     | Address Resolution Protocol        | 地址解析协议    | 已知IP地址 →→ 解析MAC地址。        |
| **RARP**    | Reverse ARP                        | 反向地址解析协议  | 已知MAC地址 →→ 解析IP地址（现已少用）。  |
| **ICMP**    | Internet Control Message Protocol  | 网际控制报文协议  | Ping命令的基础，报告差错和网络探测。      |
| **IGMP**    | Internet Group Management Protocol | 网际组管理协议   | 管理多播（组播）组成员。              |
| **RIP**     | Routing Information Protocol       | 路由信息协议    | 基于距离（跳数），适合小型网络（IGP）。     |
| **OSPF**    | Open Shortest Path First           | 开放式最短路径优先 | 基于链路状态，计算最短路，适合大型网络（IGP）。 |
| **BGP**     | Border Gateway Protocol            | 边界网关协议    | 自治系统（AS）之间的路由选择（EGP）。     |
| **NAT**     | Network Address Translation        | 网络地址转换    | 私网IP转公网IP，解决IP短缺，隐藏内网。    |
| **CIDR**    | Classless Inter-Domain Routing     | 无类别域间路由   | 利用子网掩码灵活划分子网，打破ABC类限制。    |
| LS          |                                    | 链路状态选择算法  |                           |
| DV          |                                    | 距离矢量算法    |                           |

### 五、 传输层 (Transport Layer)

| 缩写       | 英文全称                                 | 中文全称    | 核心用途与含义                    |
| :------- | :----------------------------------- | :------ | :------------------------- |
| **TCP**  | Transmission Control Protocol        | 传输控制协议  | **可靠**、面向连接（三次握手），用于文件、网页。 |
| **UDP**  | User Datagram Protocol               | 用户数据报协议 | **不可靠**、无连接、快，用于直播、游戏。     |
| **MSS**  | Maximum Segment Size                 | 最大报文段长度 | TCP数据部分的最大长度。              |
| **MSL**  | Maximum Segment Lifetime             | 最长报文段寿命 | 报文在网络存活极限，TIME_WAIT时长参考。   |
| **SCTP** | Stream Control Transmission Protocol | 流控制传输协议 | 兼顾可靠与实时，支持多流多宿主，电信信令常用。    |

### 六、 应用层 (Application Layer)

|缩写|英文全称|中文全称|核心用途与含义|
|:--|:--|:--|:--|
|**DNS**|Domain Name System|域名系统|域名 →→ IP地址（互联网电话本）。|
|**HTTP**|HyperText Transfer Protocol|超文本传输协议|网页浏览基础（80端口）。|
|**HTTPS**|HTTP Secure|安全超文本传输协议|加密的HTTP（443端口），用于支付、网银。|
|**FTP**|File Transfer Protocol|文件传输协议|传大文件，双连接（控制+数据），可靠。|
|**TFTP**|Trivial File Transfer Protocol|简单文件传输协议|传小文件，基于UDP，无连接。|
|**SMTP**|Simple Mail Transfer Protocol|简单邮件传输协议|用于**发送**邮件。|
|**POP3**|Post Office Protocol 3|邮局协议版本3|用于**接收**邮件（离线模式，下载后通常删除）。|
|**IMAP**|Internet Message Access Protocol|互联网消息访问协议|用于**接收**邮件（在线模式，服务器管理）。|
|**Telnet**|Teletype Network|远程终端协议|远程登录，明文传输（不安全）。|
|**SSH**|Secure Shell|安全外壳协议|远程登录，加密传输（安全替代Telnet）。|
|**DHCP**|Dynamic Host Configuration Protocol|动态主机配置协议|自动分配IP、网关、掩码等参数。|

### 七、 网络安全 (Network Security)

|缩写|英文全称|中文全称|核心用途与含义|
|:--|:--|:--|:--|
|**AES**|Advanced Encryption Standard|高级加密标准|对称加密算法，速度快，安全性高。|
|**RSA**|RSA Algorithm|RSA算法|非对称加密（公钥/私钥），用于签名和密钥交换。|
|**DDoS**|Distributed Denial of Service|分布式拒绝服务攻击|用海量流量淹没服务器，使其瘫痪。|
|**VPN**|Virtual Private Network|虚拟专用网络|公网上的加密专用通道，用于远程办公。|
|**FW**|Firewall|防火墙|访问控制设备，隔离内外网，过滤数据流。|
