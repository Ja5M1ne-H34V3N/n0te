# 二分搜索

所有者: H34V3N

在一列数里有 1 2 3 4 5 6 7 8按顺序排好的数，我们需要查找某个数的位置

最直接的方法，记录每个数的下标 通过遍历找到那个数 但是每到一个数 我们都要执行对比的语句 导致时间复杂度是n

利用二分搜索 能大大提高搜索效率

规则：

我们设定好最初的最大值（数组最大下标） ，最小值（0） 取其中间看其数值：

如果我们查得的数字a 大于目标数值 则在这个数之后的数都比a大 也就比目标大，这意味着这之后的数不可能是目标

这时候 我们缩小范围 改范围的最大值为这个中间数 （小于目标值 同理改最小值）

接着 我们重新计算新的最大最小值的平均值 重复操作 知道找到目标

代码示例

```python
nums = [1, 2, 3, 4, 5, 6, 7]
target = 3
low = 0
high = len(nums) - 1

while True:
    mid = (low + high) // 2
    if nums[mid] > target:
        high = mid - 1
    elif nums[mid] < target:
        low = mid + 1
    else:
        break
print(mid + "为所求")
```
