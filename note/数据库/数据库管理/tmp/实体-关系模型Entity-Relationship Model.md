# 实体-关系模型Entity-Relationship Model


一个关系模型记录了下面的几个信息：

**（身高，长度，准确率，增长率，吞吐量）**

你可能不明白，能记录这几个八竿子打不着的东西要干什么，这东西像描述一个人的，因为有身高，好像是描述物体的，因为有长度，也好像是描述货运的，因为有吞吐量…… 

# 实体-关系模型（E-R模型）

一种关系模型能包含任何类型，你想往里塞什么东西都可以，但我们在关系模型的基础上，再增加一些限制，要求记录的东西，是一个现实中客观存在的事物或对象的特征，我们称这一类关系模型叫实体-关系模型

## 实体Entity

现实中客观存在的事物或对象
