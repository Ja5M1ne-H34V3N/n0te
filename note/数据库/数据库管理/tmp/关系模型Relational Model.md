# 记录型模型record-based model

记录型模型是一类数据模型，他们在正式保存信息前，会得到一套数据模板，这些模板规定了一项数据都要包含哪些信息，这些信息的保存格式是什么，在信息进行录入的时候，信息严格按照提供的格式保存

比如一个有关学生的模板：学生(学号: 字符串(10), 姓名: 字符串(8), 年龄: 整数, 专业: 字符串(20))

包含了这个学生有且仅有哪些信息，这些信息的格式

现在，我们通过这个模板给出一个实例：

李华的信息：（1234567890，李华 ，19，计算机科学与技术）

整个记录学生信息的数据库都会按这种格式保存，他们被规定好了自己该保存什么信息，这些信息该怎么保存，就好像给出了一个表格，等待被填满，我们自己根据实际情况填补表格需要的空缺，就像是根据表格的要求“输入”信息一样。这也是**记录**的含义

## 涉及到的名词

### 记录

这里是名词的记录，表示记录型数据库的一个特定好的存储单元

比如前面例子的（1234567890，李华 ，19，计算机科学与技术）

### 记录型

这个也是名词，表示形成记录的模板，对记录的结构的定义，根据前面的描述，指：(学号: 字符串(10), 姓名: 字符串(8), 年龄: 整数, 专业: 字符串(20))
