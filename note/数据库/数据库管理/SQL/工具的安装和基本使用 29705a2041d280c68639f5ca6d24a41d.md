我们（至少是24届）使用的是pgAdmin 4和 SQL shell来进行SQL的各种操作 pgadmin 可以直接在微软的应用商店下载，安装好后 SQL shell 也有 平时打开直接按win键搜索这两个东西就好了

我们打开pgAdmin 4
